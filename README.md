# BITVIS 作业2 & 期末大作业
北京理工大学大数据可视化与可视分析课程作业
关显艾1120230619

## 数据来源
世界银行人口死亡率数据集（3个关联CSV文件）：
- API_SP.DYN.IMRT.IN_DS2_en_csv_v2_2.csv：核心死亡率数据
- Metadata_Country_API_SP.DYN.IMRT.IN_DS2_en_csv_v2_2.csv：国家-地区映射
- Metadata_Indicator_API_SP.DYN.IMRT.IN_DS2_en_csv_v2_2.csv：指标说明

## 功能
- 交互式柱状图展示2021年全球各国死亡率
- 按地区筛选（亚洲、欧洲、全部）
- 鼠标悬浮显示详细信息（国家、地区、死亡率、年份）
- 平滑过渡动画

## 技术栈
- D3.js v7
- HTML / CSS / JavaScript
- GitHub Pages 部署
